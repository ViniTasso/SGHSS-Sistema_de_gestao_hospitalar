📡 Como os Containers se Comunicam

❌ ERRADO (não funciona):

# Django tentando conectar PostgreSQL
POSTGRES_HOST = 'localhost' # ❌ Busca dentro do próprio container!

# Resultado: Connection refused ❌

✅ CORRETO (funciona):

# Django conectando PostgreSQL usando nome do serviço
POSTGRES_HOST = 'db-postgres' # ✅ Docker resolve para IP do container!

# Django conectando Auth Service
AUTH_URL = 'http://authentication-service:8001' # ✅ Funciona!

# Resultado: Conexão estabelecida ✅

🔍 Como o Docker Resolve os Nomes:

1. Django tenta conectar: db-postgres:5432
2. Docker DNS resolve: db-postgres → 172.18.0.3 (IP do container)
3. Conexão estabelecida com sucesso! ✅
